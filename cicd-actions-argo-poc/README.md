# CI/CD POC with GitHub Actions and ArgoCD

## Overview
This POC demonstrates a complete CI/CD pipeline using GitHub Actions for CI (Continuous Integration) and ArgoCD for CD (Continuous Deployment).

## Structure
```
cicd-actions-argo-poc/
├── .github/workflows/     # GitHub Actions workflows
├── test-app/             # Simple test application  
├── STEPS.md              # Step-by-step implementation guide
├── THEORY.md             # CI/CD concepts and theory
└── README.md             # This file
```

## Getting Started
1. Read `THEORY.md` for concepts
2. Follow `STEPS.md` for implementation
3. Test with the sample application in `test-app/`
